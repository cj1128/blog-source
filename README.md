# CJ's Blog

Source code of my [blog](https://cjdev.me), based on Bun, Astro and [dante-astro-theme](https://github.com/JustGoodUI/dante-astro-theme).

## Setup

- Bun v1

```bash
$ bun i
$ bun dev
```

## Routes

- `/archive/` post list
- `/` home list
- `/YYYY/MM/DD/slug/` post
  - use fixed timezone UTC+8
